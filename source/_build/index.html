编码

这一章节主要解决GameMaker Studio 2中的编码问题。 你在项目中使用的语言和方法取决于你的技能背景，为了让不同的人都能适应GameMaker Studio 2 提供了两种创建游戏的选项：

Drag and Drop (DnD™)是初学者或者那些艺术导向的 人理想的方式（也被称为可视化编程语言）。它包括了拖拽和放置动作到对象编辑器中， 以创建对象将会使用的一系列函数。你可以从下面的章节找到它的使用方法：
  1. Drag And Drop 总览
  2. Drag And Drop 参考

通过实例和程序结构，你将会对 GML 语言的语法有基本认识， 你也可以找到所有 GML 函数的指导。如果你是第一次用脚本语言，建议你从纵览部分开始，或者你可以 直接找到你需要的函数：
  1. GML 总览
  2. GML 参考

请注意这两部分不矛盾，你可以在项目中混用这两种方式，这取决于你的技能和意愿。
